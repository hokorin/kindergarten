保育のこと

みどり保育園の保育内容をご紹介します

年齢別保育内容

0〜1歳児クラス（ひよこ組）

一人ひとりの生活リズムを大切にし、安心できる環境の中で愛情豊かに保育します。 個別対応を重視し、食事・睡眠・遊びを通して基本的生活習慣の基礎を育みます。

主な活動

  • • 感覚遊び（音・光・触感）
  • • 運動発達を促す遊び
  • • 絵本の読み聞かせ
  • • 手遊び・わらべうた

発達目標

  • • 基本的信頼関係の形成
  • • 運動機能の発達
  • • 言語の芽生え
  • • 情緒の安定

2歳児クラス（りす組）

自我が芽生える時期として、一人ひとりの思いを受け止めながら、 友達との関わりを通して社会性の基礎を育てます。

主な活動

  • • 戸外遊び・散歩
  • • 簡単な製作活動
  • • 模倣遊び・ごっこ遊び
  • • 音楽リズム遊び

発達目標

  • • 自立心の芽生え
  • • 言葉の発達
  • • 友達への関心
  • • 基本的生活習慣の確立

3歳児クラス（うさぎ組）

集団生活に慣れ親しみ、友達と一緒に遊ぶ楽しさを知る時期です。 基本的生活習慣を身につけながら、様々な体験を通して豊かな感性を育みます。

主な活動

  • • 集団遊び・ルールのある遊び
  • • 絵画・製作活動
  • • 体操・運動遊び
  • • 自然観察・園外保育

発達目標

  • • 集団生活への適応
  • • 協調性の育成
  • • 創造性の発達
  • • 運動能力の向上

4〜5歳児クラス（きりん・ぞう組）

就学に向けて、文字や数への関心を高め、思考力や表現力を育てます。 責任感や協力する心を養い、リーダーシップも身につけていきます。

主な活動

  • • 文字・数遊び
  • • 科学遊び・実験
  • • 合奏・合唱
  • • クッキング活動

発達目標

  • • 就学への準備
  • • 思考力・判断力の育成
  • • リーダーシップの発揮
  • • 協力・助け合いの心

特別プログラム

音楽教室

専門講師による楽器演奏や歌唱指導で、音感とリズム感を育てます。

  • • ピアニカ演奏
  • • 合唱・合奏
  • • リトミック

農園体験

野菜の栽培から収穫まで体験し、食べ物の大切さを学びます。

  • • 季節野菜の栽培
  • • 収穫体験
  • • クッキング活動

創作活動

様々な素材を使った製作活動で、創造力と表現力を伸ばします。

  • • 絵画・工作
  • • 季節の製作
  • • 共同制作
